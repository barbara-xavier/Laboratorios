# M-todo-Insertion
Ordenação Insertion

## Relatório
  
 O for percorre o vetor da esquerda para a direita, considerando cada elemento como o próximo a ser inserido no subvetor ordenado à esquerda.
  
  - O While compara o elemento atual com os elementos do subvetor ordenado à esquerda, deslocando os elementos maiores para a direita até encontrar a posição correta.
  - Encontrando a correta, faz a troca de vallores.
  
  No main:
   - recebe um vetor desordenado.
   - chama a função 'Insertion' e imprime na tela.
